# LINE 官方帳號預約管理系統 (LINE Reservation System)

這是一套專為 LINE 官方帳號量身打造的強大預約管理系統，全站採用獨特的 **新粗野主義 / 漫畫風 (Neo-brutalism)** 介面設計，色彩鮮豔、極具互動性。系統結合 LINE LIFF 實現與使用者的無縫對接，並賦予管理者全面、彈性且高度自動化的營運控管能力！

---

## 🚀 系統特色與詳細功能說明

### 1. 📱 前台客戶端預約介面 (Client Side Booking Interface)
客戶無需下載任何 App 或註冊帳號，直接在 LINE 官方帳號點擊「預約」選單，即可自動登入並開始預約流程。
*   **無縫整合登入**：透過 LINE LIFF 自動獲取用戶基本資料，免去繁瑣的註冊流程。
*   **視覺化日曆與時段篩選**：
    *   提供直覺的日曆介面，自動隱藏未開放、或已額滿的日期。
    *   點選日期後，會即時顯示剩餘可預約的時段與項目。
*   **智慧防呆與次數限制**：
    *   **次數限制防護**：若管理員針對特定項目設定了「每人預約次數上限」，當用戶超過次數時按鈕會反灰，並提示「已達預約上限」，避免佔用資源。
    *   **衝突檢查**：若使用者在同一天已經預約了相同項目，系統也會阻擋重複預約。
    *   **黑名單與停權**：若管理員在後台將用戶停權，該使用者將無法進行新的預約。
*   **彈性資料填寫**：支援「幫小孩或他人代訂」的情境，用戶可以手動輸入實際預約者的名稱與聯絡電話，確保管理者能清楚辨識對象。
*   **歷史紀錄與自我管理**：
    *   用戶可隨時查看過去與未來的預約紀錄。
    *   針對尚未審核的預約，用戶可自行「取消預約」；但若預約已被管理員「已確認」，則無法單方面刪除，有效防止誤刪已成立的訂單。

---

### 2. 💻 管理員後台 (Admin Dashboard)
後台採用極具特色的漫畫風設計，包含粗黑框線、鮮豔色塊與立體陰影，讓管理工作也充滿樂趣。

#### 📅 預約總覽與審核 (Reservations)
*   **三態分類面板**：將預約單清楚劃分為「待審核」、「已確認」、「已取消」三大清單，一目了然。
*   **一鍵審核機制**：管理員可以快速核准或退回預約。當審核通過時，系統會**自動透過 LINE 推播「預約成功」的客製化 Flex Message** 通知用戶。
*   **視覺化農民曆**：日曆元件支援顯示「台灣農民曆」與「節氣」，並以紅字標示週末，貼合在地使用習慣。
*   **資料匯出 (Export)**：支援將預約資料一鍵匯出為 Excel (`.xlsx`) 格式，方便線下建檔或統計分析。

#### ⚙️ 強大的排班與時段設定 (Availability)
*   **預約項目字典**：可建立多個不同的預約項目（例如：體驗課程、一對一諮詢），並自訂項目名稱、標籤顏色、以及「每人預約次數上限」。
*   **彈性日曆排班**：選定特定日期，即可為當天的各個時段指派開放的項目與名額。
*   **✨ 快速複製排班**：為了減輕管理負擔，系統支援將某一天的排班設定，「一鍵複製」到當月份的其他特定星期（例如：將今天的班表複製到本月所有的週一到週五），大幅提升效率！

#### 🤖 智慧時段自動分配 (Smart Auto-Scheduling)
這正是本系統的核心亮點！專為解決「多人搶同時段」與「人工排班衝突」的痛點而生：
*   **一鍵自動運算 (Backtracking Algorithm)**：當有大量待審核訂單時，系統內建溯源演算法。只需點擊「一鍵自動排班」，系統會自動為所有待審核的預約者尋找最佳的時段配置，確保最多人能排入且時段不衝突。
*   **視覺化拖曳 (Drag & Drop)**：
    *   所有預約者會以網格呈現。若發生時段衝突，該格子會亮起**紅燈**並顯示「衝突，請拖曳移開」。
    *   管理員可以直接用滑鼠拖曳使用者的卡片，將其移動到其他空置的時段，直到所有衝突解決（亮起**綠燈**最佳狀態）。
*   **一鍵發布與推播**：排班確認無誤後，只需按下發布，系統會自動將網格內的使用者轉為「已確認」，落空的使用者轉為「取消」，並瞬間發送客製化的 LINE 推播給所有人！

#### 👥 用戶管理與行銷 (User Management)
*   **自動建檔客戶庫**：自動記錄所有透過 LINE 登入過系統的用戶。
*   **客製化顯示名稱**：管理員可修改使用者的「顯示名稱」（例如改為小孩姓名），系統將鎖定該名稱，不受使用者未來更改 LINE 暱稱的影響。
*   **多維度標籤與過濾**：
    *   支援為用戶設定「年齡群組」、「性別」、「星座」、「專屬自訂標籤」（如：VIP、黑名單）與「內部備註」。
    *   提供強大的多重篩選器，能精準找出目標客群（例如：篩選出特定星座且具有 VIP 標籤的女性）。
*   **批次推播與一對一發送**：勾選過濾出來的客群後，即可發送批次客製化 LINE 推播，是進行精準行銷、重要公告的絕佳工具。

#### 📝 訊息範本與富文本編輯 (Message Settings)
*   **客製化圖文推播 (Flex Message)**：不論是預約成功、審核拒絕或是自訂發送，系統皆採用精美的 LINE Flex Message 卡片。支援上傳預設橫幅圖片 (JPG/PNG)，讓訊息圖文並茂。
*   **✨ 富文本編輯器 (Rich Text Editor)**：
    *   內建所見即所得的編輯器，支援設定文字的**大小**、**字體顏色**、**粗體**，以及段落的**靠左/置中/靠右/分散對齊**。
    *   **插入表情符號**：內建繁體中文化的 Emoji 面板，可輕鬆在推播內文中插入各種生動的表情符號！
*   **常用訊息庫 (Quick Replies)**：
    *   支援建立常用的罐頭訊息，並自動編排「序號」。
    *   **場景分類機制**：可設定該常用訊息適用於「預約成功」、「審核通知」或「自訂推播」等場景。當管理員編輯對應欄位時，系統會智慧過濾出專屬該場景的常用訊息，一鍵點擊即可帶入內容，免去重複打字的麻煩。
*   **動態變數支援**：主標題與內文皆支援使用變數 `{好友的顯示名稱}`，發送時系統會自動替換為該用戶的真實稱呼，增加親切感。

#### 🔐 系統設定 (System Settings)
*   支援在前端介面直接設定與切換 LINE Channel Access Token 等 API 金鑰，不需修改後台環境變數即可動態綁定不同的 LINE 官方帳號。

## 🛠️ 技術架構
- **前端框架**: [React 19](https://react.dev/) + [Vite](https://vitejs.dev/)
- **資料庫與後端**: [Firebase (Firestore, Storage, Auth)](https://firebase.google.com/) + Vercel Serverless Functions (`/api/send-line-message`)
- **樣式與 UI**: [Tailwind CSS](https://tailwindcss.com/) (Neo-brutalism 漫畫風設計) + [Lucide React](https://lucide.dev/) (Icons) + React Quill New (富文本編輯) + Emoji Picker React
- **時間與農民曆**: [date-fns](https://date-fns.org/) + [lunar-javascript](https://github.com/6tail/lunar-javascript)
- **資料匯出**: [SheetJS (xlsx)](https://sheetjs.com/)

## 💻 本地開發與運行
1. **安裝依賴套件** (因部分舊套件相依性，建議使用 legacy-peer-deps)
   ```bash
   npm install --legacy-peer-deps
   ```

2. **設定環境變數**
   請參考 `.env.example`，建立 `.env` 檔案，並填入您的 Firebase 與 LINE API 金鑰。

3. **啟動開發伺服器**
   ```bash
   npm run dev
   ```

4. **編譯打包 (供 Vercel 或其他靜態伺服器部署)**
   ```bash
   npm run build
   ```
